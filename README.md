# 2DImageTransform
세종대학교 2021 동계 창의학기제로 진행한 2D 이미지 변환 탐구 프로그램입니다.

이미지 불러오기 혹은 웹캠 캡쳐를 통해 가져온 이미지를 이미지 변환할 수 있습니다.
